# CONTRIBUTING

Thank you for your interest in contributing to WISDOM.

WISDOM is an operational governance framework designed to remain coherent, use-case driven, and lightweight. Contributions are welcome as long as they preserve that philosophy.

---

## How to Contribute

1. Fork the repository
2. Create a dedicated branch
3. Make your changes clearly
4. Submit a Pull Request explaining:

   * What you changed
   * Why it improves clarity or operational value

All contributions are reviewed before being merged.

---

## What Contributions Are Welcome

* Real-world use case examples
* Improvements in clarity or structure
* Refinements to safeguard playbooks
* Sector-specific adaptations
* Documentation improvements
* Translations

---

## Scope Protection

The six Safeguards and the core architecture form the structural foundation of WISDOM.

Significant structural changes should be discussed before submission to ensure consistency.

WISDOM favors clarity and proportionate governance over complexity.

---

## Attribution

Contributors are automatically credited through GitHub commit history and Pull Requests.
